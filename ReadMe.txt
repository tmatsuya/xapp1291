2. OVERVIEW

This readme describes how to use the files that come with XAPP1291 

The files and usage of the files is described in XAPP1291.


3. SOFTWARE TOOLS AND SYSTEM REQUIREMENTS

Vivado 2016.2 design tools, Software Development Kit (SDK)

4. DESIGN FILE HIERARCHY

  C:\xapp1291\ReadMe.txt - this file

  C:\xapp1291\design\ directory -
    Contains the script that generates the bit stream file
    See xapp1291 for usage instructions.

  C:\xapp1291\ready_for_download\ directory -
    Contains the pre-built bit stream and application binary file along with script to 
	download them to the board. See xapp1291 for usage.

  C:\xapp1291\sdk\ directory -
    The SDK project directory that contains the application reference design and script to 
	create the sdk project. See xapp1291 for usage instructions.

5. INSTALLATION AND OPERATING INSTRUCTIONS 

  Unzip the xapp1291.zip archive directory tree to the root C:\ drive on a Windows PC that is supported by the Xilinx SDK.

  Begin by reviewing the xapp1291 for instructions.
